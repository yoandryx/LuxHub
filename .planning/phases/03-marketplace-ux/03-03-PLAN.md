Implementation details:

**Dropzone area:**
- `useDropzone({ accept: { 'image/jpeg': [], 'image/png': [] }, maxSize: maxSizeMB * 1024 * 1024, multiple: true, onDrop })`
- Dropzone div: dashed border, `2px dashed var(--lux-border)`, `border-radius: var(--lux-radius-lg)`, padding 32px, center-aligned
- On drag active: `border-color: var(--lux-accent)`, `background: var(--lux-accent-glow)`
- Icon: `HiOutlinePhotograph` 48px, `color: var(--lux-text-muted)`
- Text: "Drag photos here or click to browse" (14px)
- Subtext: "Upload at least 5 photos. JPG or PNG, max 10MB each." (12px, `color: var(--lux-text-secondary)`)

**onDrop handler:**
- For each accepted file, create a local preview via `URL.createObjectURL(file)`
- Create an `UploadedImage` entry with `uploading: true`, `progress: 0`, preview URL
- Add all entries to images array via `onChange`
- Upload each file to `/api/arweave/upload` using `fetch` with `FormData`
- Use `p-limit` (check if installed, otherwise just use a simple queue of 3 concurrent uploads via a counter)
- Actually, simpler: use `Promise.all` on the array â€” the upload endpoint handles one at a time, and 5-10 parallel fetches is fine
- On upload success: update the entry's `url` to the returned URL, `uploading: false`, `progress: 100`
- On upload error: set `error: 'Upload failed. Check your connection and try again.'`
- Maintain array ORDER: `Promise.all` preserves input order

**Preview grid (with @dnd-kit reordering):**
- `DndContext` with `closestCenter` collision detection and `onDragEnd` handler
- `SortableContext` with `rectSortingStrategy` wrapping the grid
- Grid: `display: grid`, `grid-template-columns: repeat(auto-fill, 90px)`, `gap: 8px`, `margin-top: 16px`
- Each item is a `SortableImage` sub-component using `useSortable` hook:
  ```typescript

    const { attributes, listeners, setNodeRef, transform, transition } = useSortable({ id: image.id });
    const style = { transform: CSS.Transform.toString(transform), transition };
    // Render thumbnail with delete button, progress bar, and "Primary" badge if isFirst
  }
  ```
- Each thumbnail: 80x80px, `border-radius: var(--lux-radius-sm)`, `object-fit: cover`
- Delete button: 20px circle, top-right corner, `var(--lux-error)` background, white X icon, `onClick: onRemove`
- Upload progress bar: thin (3px) bar below thumbnail, `var(--lux-accent)` fill, width = `${progress}%`
- First image badge: "Primary" label, 10px font, `var(--lux-accent)` background with white text, positioned bottom-left
- Drag handle: the entire thumbnail is draggable (via `{...attributes} {...listeners}`)
- `onDragEnd`: call `onChange(arrayMove(images, oldIndex, newIndex))`

**Cleanup:** Revoke preview blob URLs when component unmounts or images change (prevent memory leaks): `useEffect` cleanup that calls `URL.revokeObjectURL` on removed previews
