Implementation — replicate MobileDrawer's drawer shell pattern:
- **Overlay**: `position: fixed`, `inset: 0`, `background: rgba(0, 0, 0, 0.5)`, `z-index: 999`, `onClick: onClose`
- **Drawer panel**: `position: fixed`, `top: 0`, `left: 0`, `width: 85vw`, `max-width: 360px`, `height: 100vh`, `z-index: 1000`
- **Slide-in animation**: CSS class `.drawer` with `animation: drawerSlideIn 0.3s ease-out` on mount
- **Glass-morphism**: `background: var(--lux-glass, rgba(13, 13, 13, 0.85))`, `backdrop-filter: blur(20px)`, `-webkit-backdrop-filter: blur(20px)`
- **Header**: flex row with title text and close button (HiOutlineX, 44px touch target)
- **Body scroll lock**: `useEffect` that sets `document.body.style.overflow = 'hidden'` when `isOpen` and restores on cleanup
- **Escape key**: `useEffect` with keydown listener for Escape key -> `onClose()`
- **Scrollable content area**: `overflow-y: auto`, `-webkit-overflow-scrolling: touch`, padding 16px
- Render `{children}` in the scrollable content area
- Only render overlay + drawer when `isOpen === true`
