    ```typescript

      payerKey: payer.publicKey,
      recentBlockhash: blockhash,
      instructions,
    }).compileToV0Message();
    const transaction = new VersionedTransaction(txMessage);
    transaction.sign([payer]);
    const signature = await connection.sendTransaction(transaction, { skipPreflight: false });
    await connection.confirmTransaction(signature, 'confirmed');
    ```

    Replace with:
    ```typescript
    import { sendWithRetry } from '@/lib/solana/retryTransaction';
    ```

    The change is specifically replacing lines ~182-195 (the outer transaction send) with the single `sendWithRetry` call, passing the `instructions` array (which contains vaultTxCreateIx, proposalCreateIx, and optionally proposalApproveIx).
