## API Endpoints Summary

### Pool Management

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/pool/convert-from-escrow` | POST | Convert listing to pool with auto-tokenization |
| `/api/pool/invest` | POST | Buy shares in open pool |
| `/api/pool/custody` | POST | Update custody status |
| `/api/pool/pay-vendor` | POST | Release funds after custody verified |
| `/api/pool/distribute` | POST | Distribute proceeds to token holders |
| `/api/pool/list` | GET | List pools with new fields |
| `/api/pool/[id]` | GET | Get pool details |

### Bags Integration

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/bags/create-pool-token` | POST | Create pool share token |
| `/api/bags/configure-fee-share` | POST | Set up 3% fee routing |
| `/api/bags/trade-quote` | GET | Get swap quote |
| `/api/bags/execute-trade` | POST | Execute token swap |
| `/api/bags/partner-stats` | GET | Get partner earnings |
